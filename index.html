SOMETHING VERY MOTIVATIONAL

Данный сайт был создан с целью применения изученных базовых тегов HTML и свойств CSS

Планирую применить к тексту какой-нибудь сносный красивый фон, найти в интернете готовую поработать с цветовой палитрой и просто ввести параметр "border-radius: 50%;" закруглить края фото выше. Гениально, правда?

Что-то вовсе бесполезное. Вы знали, что в 2012 году Охлобыстин хотел баллотироваться в президенты? С ума сойти.

HTML
"HyperText Markup Language", язык гипертекстовой разметки, служащий для создания изначального макета
CSS
"Cascad StyleSheet", каскадная таблица стилей, применяемая для придания сайту определённого вида

Пример кода на HTML:


<body>
<p>Новый абзац с текстом</p>
<img src="https://images/image.png">
</body>

Пример кода на CSS:


body {
   font-style: "Courier New", monospace;
   font-size: 12px;
  background-color: rgb(23, 45, 90);
}

.code {
   font-decoration: line-through;
   font-weight: bold;
}
